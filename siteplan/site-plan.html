Individual Project Site Plan

Site Name

Green Trails
This name was selected because the website focuses on eco-friendly hiking trails and nature walks. It evokes nature and sustainability.

Site Purpose

The site is a resource hub for individuals seeking nature trails, eco-hikes, and environmentally friendly outdoor activities. It provides trail reviews, eco-tips, and a contact form for trail submissions.

Scenarios

  • Where can I find beginner-friendly eco-hiking trails in my area?
  • How can I share a new trail I've discovered with the community?

Color Schema

  • Forest Green (#006400) – Used for headings, accents, and borders.
  • Mint Cream (#f4fff4) – Used for the background color to create a calm, nature-inspired vibe.

Typography

  • Playfair Display – Used for main headings and section headers to create an elegant contrast.
  • Roboto – Used for body text for its readability and modern aesthetic.
